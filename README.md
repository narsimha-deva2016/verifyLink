A fun and interactive Valentine's Day survey app built with React!

## Features

- **Welcome Page**: Cute greeting with a button to start
- **Question Page**: Valentine proposal with a moving "No" button
- **Success Page**: Beautiful celebration page with floating hearts

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## How It Works

1. **Welcome Page**: Greets your special someone
2. **Question Page**: Asks "Will you be my Valentine?"
   - "Yes" button leads to success
   - "No" button moves around and changes text when hovered/clicked
3. **Final Page**: Shows a beautiful page with hearts and love messages
